## ICC Men’s T20 World Cup 2024 Analysis – Power BI Dashboard

### Objective
To analyze team and player performance in the ICC Men’s T20 World Cup using interactive visualizations.

### Key Metrics
- Team Rankings
- Highest Run Scorers
- Most Sixes by Batsmen
- Highest Wicket Takers
- Team-wise Winning Percentage
- Batting Average Comparison

### Key Insights
- India finished at the top position in the tournament
- Certain batsmen dominated both runs and boundary hitting
- Bowlers played a crucial role with consistent wicket-taking
- Winning percentage varies significantly among teams

### Tools Used
- Power BI
- DAX Measures
- Data Cleaning & Modeling
