## Project Overview
This project is an interactive **Sales Dashboard built using Microsoft Excel**.  
It provides clear insights into sales performance using charts, pivot tables, and structured analysis.

## Objectives
- Analyze sales data efficiently
- Identify sales trends and performance
- Create a user-friendly dashboard for decision-making

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Data Cleaning & Formatting
- Dashboard Design

## Dataset Information
- Sheet Name: `sales_data`
- Contains raw sales-related data used for analysis and reporting

## Analysis Sheet
- Data transformation
- Calculations for meaningful insights
- Structured analysis before visualization

## Dashboard Sheet
- Interactive charts
- Visual representation of sales trends
- Easy-to-understand layout for business users

## How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Navigate to the **Dashboard** sheet
4. Use filters (if available) to explore insights
